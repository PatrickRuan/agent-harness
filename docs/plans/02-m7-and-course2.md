# 擴充教材：新增 M7（OpenClaw 拆解）＋ 開設第二門課（AI 產業與社會）

## Context（為什麼做這個）

使用者提供了一份 22 頁簡報《給 Claude 的(I) 瘋狂遊戲》，詢問內容該併入現有六堂課，還是另開新課程。

讀完後判斷：**這份簡報自然裂成性質完全不同的兩半**，因此採取拆分處理（已與使用者確認）：

- **p17–22（OpenClaw）** 是現有課程核心論點的**最佳實證**。簡報中「OpenClaw 沒有 AI，他是一個調度流程與執行的程式」這句話就是 harness 的定義；p19 的「說明書」（System Prompt + Identity.md + User.md + Memory.md，**加入每一次的對答中**）正是 M1 教的「模型沒有記憶，每輪重寄整個箱子」的真實案例。現有課程只在 M5 把 OpenClaw 當趨勢帶過，缺的正是「拿真實產品解剖一次」的壓軸。→ **併入現有課程成為 M7。**
- **p1–16（產業鏈、地緣、就業崩潰、UHI、存在焦慮）** 回答的是完全不同的問題，且大量內容是**推測與個人觀點**（如匿名來源的「2028/6 金融體系毀滅」）。現有課程刻意建立在可查證事實上，混入推測會讓學生用同樣信任度看待兩者，傷害既有可信度。→ **同 repo 新資料夾開第二門課，明確標示證據等級。**

受眾同前：頂大電機系大一新生，聰明但不預設程式背景，白話優先。

## 決策（已與使用者確認）

1. **拆分**：OpenClaw → M7 併入課程一；產業與社會 → `course2/` 獨立第二門課。

## 實作要點

- 重用既有工具：`Lab.stepper`、`Lab.toggler`、`Lab.el`、`Lab.clear`（見 `assets/app.js`）
- 所有 DOM 一律 `createElement` + `textContent`，不使用 `innerHTML`
- 沿用既有驗證過的修正：不使用 `backdrop-filter`、不使用全域 `scroll-behavior:smooth`
- 圖示一律 inline SVG，顏色走 CSS 變數以支援深淺色；同頁多個 SVG 的 `<marker id>` 不可重複
- 數字一律由程式計算，不在文案裡寫死（前次踩過的坑）
